SACNR Monitor Python API (By PBomb)
====================================

SACNR Website:         http://www.sacnr.com
SACNR Monitor Website: http://monitor.sacnr.com


Requirements:
--------------

	* Python 2.7

NOTE: Newer/Older Versions of python could pontentially work
      But have not been tested).

Installation:
--------------

You have serveral options to install the module.

	* Run "python setup install" (or "setup.py install" 
	  depending on OS) from source distribution

	* Copy 'SANCRMonitor.py' to appropriate Python Lib directory

Example:
---------

	* SACNRMonitor-example.py
		A simple example of how to use the SACNRMonitor module.
